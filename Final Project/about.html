About Us

Hive Scheduler is a scheduling app created for restaurants and businesses wherein managers and employees can access their schedules. Hive Scheduler provides managers and employees with two different views. Managers can view their employees, add new employees to the system, edit schedules and assign shifts to their staff. Employees can edit their user account, add their availability, request for time off as well as view their schedule.

How does it work?

We've created this web app using Java for the backend, mySQL to store information on the database and HTML/CSS/JS.
